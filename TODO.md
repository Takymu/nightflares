# NightFlares — что осталось доделать

Всё, что помечено на сайте плашкой **«заглушка»** или **⚠**, ждёт реальных данных.

## 1. Фото под замену (класть в `img/`, имена ровно такие; можно кинуть `.jpg` — пережму в webp)

| Файл | Что снять | Где на сайте |
|---|---|---|
| `img/mushroom-1.jpg` | Ночник **«Гриб»** (когда будет) | карточка «Гриб» |
| `img/bird-soon.jpg` | **«Птица»** — фото/эскиз/заготовка | карточка «Птица» |
| `img/process-concept.jpg` | Скриншот из **Blender** | «Как делается», шаг 1 |
| `img/process-print.jpg` | Печать на принтере | шаг 2 |
| `img/process-solder.jpg` | Стол с паяльником *(снимешь)* | шаг 3 |
| `img/process-pack.jpg` | Коробка с изолоном *(снимешь)* | шаг 4 |

Как убрать плашку после подстановки фото:
- «Гриб»/«Птица» — в `script.js` у модели поставить `photoReady: true`;
- шаги процесса — в `index.html` убрать `pstep--stub` из `class="pstep pstep--stub"`.

## 2. Данные в конфиге (`script.js`, объект `CONFIG`)

- `avito` — ссылка на профиль Авито (сейчас `{{AVITO_LINK}}`, в футере подсвечена как заглушка).
- «Гриб»: `price` и `size` — когда модель будет готова.
- Проверь `tgUsername: "Takyum"` — это аккаунт для кнопки «Написать». Если писать должны в другой — поменяй.

## 3. Юридические страницы (`privacy.html`, `oferta.html`)

- В шапке каждой — пометка «показать юристу перед публикацией». **Показать юристу.**
- Осталось заполнить `{{ДАТА}}` (дата публикации). ФИО/ИНН/почта/доставка уже подставлены.

## 4. После публикации на хостинг

- `index.html` → `og:image` заменить на **абсолютный** URL, напр. `https://твойдомен/img/og-cover.jpg`
  (иначе карточка в Telegram может не подтянуть картинку). Строка помечена `<!-- TODO -->`.
- По желанию: `favicon.ico` для старых браузеров (сейчас только `favicon.svg`).
- Прогнать Lighthouse (mobile) в Chrome DevTools — цель Performance и Accessibility ≥ 90.

## Готово (менять не нужно)

- **Переключатель режимов яркости** — реальные фото `img/mode-0..3`, стартует на «Средне». Подписи/дефолт в `CONFIG.brightness`.
- Реальные фото «Вейва»: `hero-wave-lit`, `wave-1`, `wave-2`, `og-cover`.
